## Purpose

Manages the creation and maintenance of podcast feeds.

## Entities

### Feed

> `bridge/rss_transformer.py`

Represents an RSS feed configuration for a user.

**Properties:**

| Name | Type | Description |
|------|------|-------------|
| name | string | Unique identifier for the feed |
| url | string | URL of the RSS feed |
| style | string | Transformation style for the feed |

**Relationships:**

- belongs to User (Feed belongs to a specific user)
- has-many Episode (Feed contains multiple episodes)
