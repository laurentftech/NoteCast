### Requirement: GETArtifact

The API SHALL support `GET /artifacts` to fetch a list of artifacts from the notebooklm api.

**Response:**

```json
{
  "artifacts": [
    {
      "id": "string",
      "title": "string",
      "wav_url": "string",
      "created_at": "string"
    }
  ]
}
```

#### Scenario: FetchArtifacts
- **GIVEN** The NotebookLM API is available
- **WHEN** GET /artifacts is called
- **THEN** 200 OK with a list of artifacts in the response body

### Requirement: POSTEpisode

The API SHALL support `POST /episodes` to convert a wav file to mp3 and save it to the episodes directory.

**Request:**

```json
{
  "artifact_id": "string",
  "wav_url": "string",
  "title": "string"
}
```

**Response:**

```json
{
  "mp3_filename": "string",
  "created_at": "string"
}
```

#### Scenario: ConvertWAVToMP3
<!-- spec-gen-test: tags=smoke (auto) -->
- **GIVEN** A valid WAV file URL
- **WHEN** POST /episodes is called
- **THEN** 201 Created with the MP3 filename and creation date in the response body

### Requirement: DELETEEpisode

The API SHALL support `DELETE /episodes` to purge old episodes that are older than the retention period.

**Response:**

```json
{
  "purged_episodes": [
    {
      "artifact_id": "string",
      "mp3_filename": "string"
    }
  ]
}
```

#### Scenario: PurgeOldEpisodes
- **GIVEN** A history of episodes with some older than the retention period
- **WHEN** DELETE /episodes is called
- **THEN** 200 OK with a list of purged episodes in the response body

### Requirement: GETFeed

The API SHALL support `GET /feed.xml` to generate an rss feed from the history of episodes.

**Response:**

```json
{
  "rss_feed": "string"
}
```

#### Scenario: GenerateRSSFeed
- **GIVEN** A history of episodes
- **WHEN** GET /feed.xml is called
- **THEN** 200 OK with the RSS feed in the response body
